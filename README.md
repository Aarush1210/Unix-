## Password Generator – UNIX Shell Script

This project is a simple "menu-driven Password Generator" written in "UNIX shell scripting (bash)".  
It allows users to create strong random passwords, save them, view them, delete them, and even generate themed passwords.

This script is designed to be "beginner-friendly", easy to understand, and runnable on any Linux/Unix system.

---

## Features

###  1. Generate Strong Password  
Creates a random secure password of user-defined length.

###  2. Save Password  
Stores any password into a text file (`saved_passwords.txt`).

###  3. View Saved Passwords  
Displays all previously saved passwords.

###  4. Delete Saved Passwords  
Deletes the saved passwords file from the system.

###  5. Generate Themed Password  
Generate passwords in three themes:  
- Simple (A–Z, a–z)  
- Symbols  
- Numbers Only  

###  6. Exit  
Closes the program.
